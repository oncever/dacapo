Download

To use the suite, all you need is the dacapo jar, which can be downloaded here (19MB).

We have a large source distribution available here (45MB).  To build the dacapo jar from these sources you will need the apache 'ant' tool - see the supplied README file for details.

We have a smaller source distribution available here (8.8MB). This contains the DaCapo benchmark software but none of the third-party software. The ant task 'sources' downloads the required packages.
