The DaCapo Benchmark Suite

This benchmark suite is intend as a tool for the the memory management research community.  It consists of a set of open source, real world applications with non-trivial memory loads.

A beta release of the suite is currently available (currently version beta051009) for download and evaluation.

Please note: It is essential that the version number associated with the release be cited in any use of the benchmark.

Feedback

We invite feedback on our beta from the community.  Please mail us at: dacapo-benchmark@cs.umass.edu
