This benchmark suite is intended as a tool for Java benchmarking and memory management research communities. It consists of a set of open source, real world applications with non-trivial memory loads.

A beta release of the suite is currently available (currently version beta-2006-08) for download and evaluation.

Please note: It is essential that the version number associated with the release be cited in any use of the benchmark, and we ask that you please cite the 2006 OOPSLA paper describing the suite.

Feedback

We invite feedback on our beta from the community.  Please mail us at: dacapo-benchmark@cs.umass.edu

License
