This benchmark suite is intended as a tool for Java benchmarking and memory management research communities. It consists of a set of open source, real world applications with non-trivial memory loads.

A beta release of the suite is currently available (currently version beta060709) for download and evaluation.

Please note: It is essential that the version number associated with the release be cited in any use of the benchmark, and please cite the following paper (to appear).
