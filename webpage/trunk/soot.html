This page is a place holder for information on how to use the DaCapo benchmarks with ahead of time analysis tools such as Soot. This requires using the new 'xdeps' repackaging of the suite (eg dacapo-2006-10-MR2-xdeps.zip).

Please note that we strongly recommend that you use the regular jar packaging of the suite (eg dacapo-2006-10-MR2.jar) unless you specifically need to use the 'xdeps' repackaging.
